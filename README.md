# Motif - Your Personal Progression Lab

Motif is a digital workspace for musicians to create, save, and analyze chord progressions. With Motif, you can capture musical ideas, understand their harmony using automatic Roman numeral analysis, and build a personal library of original music—all in a clean, musician-friendly app.

---

## Features

- **Create Chord Progressions:** Easily build and edit chord progressions.
- **Automatic Roman Numeral Analysis:** Instantly see the harmonic function of your chords.
- **Save & Organize:** Store your progressions and organize them for future reference.
- **Clean Interface:** Designed by musicians, for musicians—no clutter, just creativity.

---

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/motif.git
   cd motif
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```bash
   npm start
   # or
   yarn start
   ```

4. Open your browser and go to `http://localhost:3000`

---

## Usage

- **Create a new progression:** Click "New Progression" and start adding chords.
- **Analyze:** The app will automatically show Roman numeral analysis for your progression.
- **Save:** Give your progression a name and save it to your library.
- **Browse:** View, edit, or delete your saved progressions anytime.
